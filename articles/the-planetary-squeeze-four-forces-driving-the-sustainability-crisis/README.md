# The Planetary Squeeze: Four Forces Driving the Sustainability Crisis

This article folder supports the Planetary Boundaries knowledge-series article on the planetary squeeze: population growth, rising affluence, climate change, and ecosystem degradation as interacting forces that narrow safe operating space.

The article body includes Python and R workflows for readability and reproducible analytics. This repository extends the implementation into a broader engineering scaffold for auditability, APIs, scenario scoring, SQL provenance, edge anomaly detection, and planetary squeeze diagnostics.

## Structure

- `python/` — planetary squeeze diagnostics
- `r/` — planetary squeeze dashboarding
- `sql/` — schema for scenarios, four-force pressure indicators, governance capacity, and audit trails
- `rust/` — reliable squeeze scoring-core scaffold
- `go/` — lightweight diagnostic API scaffold
- `c/` — embedded-style risk alert scaffold
- `cpp/` — scenario simulation scaffold
- `tinyml/` — edge anomaly detection scaffold
- `pynq/` — accelerator-oriented monitoring scaffold
- `docs/` — methodological and engineering notes
- `data/raw/` — raw input data
- `data/processed/` — cleaned and transformed data
- `outputs/` — exported diagnostic scores and dashboard-ready files

## Methodological Note

The sample workflows use illustrative values. Replace placeholder values with documented demographic data, resource-use indicators, climate metrics, ecosystem indicators, planetary-boundary status data, governance assessments, and transparent assumptions before applied use.
