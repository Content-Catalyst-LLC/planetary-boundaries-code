# Implementation Notes

This module is designed around five linked questions:

1. How much pressure is created by demographic scale?
2. How much pressure is created by affluence and material consumption?
3. How much climate stress is present?
4. How much ecosystem degradation is present?
5. How do governance, adaptive, justice, mitigation, restoration, and material-efficiency capacities reduce risk?

All values in the sample workflows are illustrative. Replace them with documented demographic data, consumption indicators, climate metrics, ecosystem indicators, planetary-boundary data, governance assessments, and transparent assumptions before applied use.
