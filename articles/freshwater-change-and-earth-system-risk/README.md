# Freshwater Change and Earth System Risk

This article folder supports the Planetary Boundaries knowledge-series article on freshwater change, blue water, green water, streamflow deviation, soil-moisture deviation, groundwater stress, hydrological extremes, wetland buffering, and Earth system risk.

The article body includes Python and R workflows for readability and reproducible analytics. This repository extends the implementation into a broader engineering scaffold for auditability, APIs, systems programming, embedded monitoring, TinyML, and PYNQ-oriented edge workflows.

## Structure

- `python/` — freshwater change and hydrological risk diagnostics
- `r/` — freshwater boundary dashboarding
- `sql/` — schema for streamflow, soil moisture, groundwater, buffers, exposure, and audit trails
- `rust/` — reliable freshwater-risk scoring-core scaffold
- `go/` — lightweight diagnostic API scaffold
- `c/` — embedded-style streamflow and soil-moisture threshold alert scaffold
- `cpp/` — freshwater policy scenario simulation scaffold
- `tinyml/` — edge anomaly detection scaffold
- `pynq/` — accelerator-oriented monitoring scaffold
- `docs/` — methodological and engineering notes
- `data/raw/` — raw input data
- `data/processed/` — cleaned and transformed data
- `outputs/` — exported diagnostic scores and dashboard-ready files

## Methodological Note

The sample workflows use illustrative data. Replace placeholder values with documented streamflow records, soil-moisture observations, groundwater data, remote-sensing products, hydrological models, environmental-flow assessments, and transparent assumptions before applied use.
