# Anthropocene Sustainable Development: Rethinking Prosperity on a Finite Planet

This article folder supports the Planetary Boundaries knowledge-series article on Anthropocene sustainable development, prosperity within limits, social foundations, planetary boundaries, governance capacity, justice capacity, resilience, and transition urgency.

The article body includes Python and R workflows for readability and reproducible analytics. This repository extends the implementation into a broader engineering scaffold for auditability, APIs, prosperity-within-limits scoring, SQL provenance, edge anomaly detection, and Anthropocene development diagnostics.

## Structure

- `python/` — Anthropocene sustainable development diagnostics
- `r/` — prosperity within limits dashboarding
- `sql/` — schema for development scenarios, social foundations, ecological pressure, and audit trails
- `rust/` — reliable prosperity scoring-core scaffold
- `go/` — lightweight diagnostic API scaffold
- `c/` — embedded-style risk alert scaffold
- `cpp/` — scenario simulation scaffold
- `tinyml/` — edge anomaly detection scaffold
- `pynq/` — accelerator-oriented monitoring scaffold
- `docs/` — methodological and engineering notes
- `data/raw/` — raw input data
- `data/processed/` — cleaned and transformed data
- `outputs/` — exported diagnostic scores and dashboard-ready files

## Methodological Note

The sample workflows use illustrative values. Replace placeholder values with documented development indicators, planetary-boundary data, ecological pressure metrics, governance indicators, justice metrics, resilience assessments, and transparent assumptions before applied use.
