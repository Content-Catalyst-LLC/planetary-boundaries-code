# Implementation Notes

This module is designed around five linked questions:

1. Are social foundations being met?
2. Is wellbeing improving?
3. Is development reducing or increasing ecological pressure?
4. Is planetary-boundary pressure within a safer range?
5. Do governance, justice, resilience, material efficiency, mitigation, and restoration capacities support durable prosperity?

All values in the sample workflows are illustrative. Replace them with documented development indicators, planetary-boundary data, ecological pressure metrics, governance indicators, justice metrics, resilience assessments, and transparent assumptions before applied use.
