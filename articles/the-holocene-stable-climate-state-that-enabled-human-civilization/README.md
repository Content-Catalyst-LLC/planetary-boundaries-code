# The Holocene: The Stable Climate State That Enabled Human Civilization

This article folder supports the Planetary Boundaries knowledge-series article on Holocene climate stability, paleoclimate baselines, ice-core evidence, agriculture, civilization, planetary boundaries, safe operating space, Anthropocene departure, and sustainable development.

The article body includes Python and R workflows for readability and reproducible analytics. This repository extends the implementation into a broader engineering scaffold for auditability, APIs, baseline scoring, SQL provenance, edge anomaly detection, and Holocene stability diagnostics.

## Structure

- `python/` — Holocene stability diagnostics
- `r/` — Holocene baseline dashboarding
- `sql/` — schema for Holocene references, observed indicators, anomalies, and audit trails
- `rust/` — reliable baseline scoring-core scaffold
- `go/` — lightweight diagnostic API scaffold
- `c/` — embedded-style departure alert scaffold
- `cpp/` — Holocene departure scenario simulation scaffold
- `tinyml/` — edge anomaly detection scaffold
- `pynq/` — accelerator-oriented monitoring scaffold
- `docs/` — methodological and engineering notes
- `data/raw/` — raw input data
- `data/processed/` — cleaned and transformed data
- `outputs/` — exported diagnostic scores and dashboard-ready files
