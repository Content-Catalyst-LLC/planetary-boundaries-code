# Implementation Notes

This module is designed around five linked questions:

1. What Holocene reference condition is being used?
2. How far has an observed indicator moved from that reference?
3. Is the departure large relative to Holocene variability?
4. Does the indicator exceed a boundary reference?
5. How do governance capacity, adaptive capacity, exposure, and cross-system interaction shape risk?

All values in the sample workflows are illustrative. Replace them with documented paleoclimate datasets, boundary estimates, scenario data, monitoring records, governance assessments, and transparent assumptions before applied use.
