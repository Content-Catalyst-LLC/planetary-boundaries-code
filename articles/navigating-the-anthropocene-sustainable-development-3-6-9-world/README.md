# Navigating the Anthropocene: Sustainable Development in a 3–6–9 World

This article folder supports the Planetary Boundaries knowledge-series article on Anthropocene planetary risk, sustainable development, the 3–6–9 heuristic, climate pressure, biosphere pressure, demographic-development demand, boundary transgression, governance capacity, justice capacity, and transformation urgency.

The article body includes Python and R workflows for readability and reproducible analytics. This repository extends the implementation into a broader engineering scaffold for auditability, APIs, scenario scoring, SQL provenance, edge anomaly detection, and Anthropocene risk diagnostics.

## Structure

- `python/` — 3–6–9 Anthropocene risk diagnostics
- `r/` — Anthropocene risk dashboarding
- `sql/` — schema for scenarios, pressures, governance capacity, and audit trails
- `rust/` — reliable scenario scoring-core scaffold
- `go/` — lightweight diagnostic API scaffold
- `c/` — embedded-style risk alert scaffold
- `cpp/` — scenario simulation scaffold
- `tinyml/` — edge anomaly detection scaffold
- `pynq/` — accelerator-oriented monitoring scaffold
- `docs/` — methodological and engineering notes
- `data/raw/` — raw input data
- `data/processed/` — cleaned and transformed data
- `outputs/` — exported diagnostic scores and dashboard-ready files

## Methodological Note

The sample workflows use illustrative values. Replace placeholder values with documented climate scenarios, biodiversity indicators, demographic projections, planetary-boundary status data, governance assessments, and transparent assumptions before applied use.
