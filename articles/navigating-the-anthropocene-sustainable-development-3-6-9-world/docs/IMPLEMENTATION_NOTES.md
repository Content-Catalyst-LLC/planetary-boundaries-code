# Implementation Notes

This module is designed around five linked questions:

1. How much climate pressure is represented in a scenario?
2. How much biosphere pressure is represented?
3. How much development demand is represented?
4. How do planetary-boundary transgressions amplify systemic risk?
5. How do governance, adaptive, justice, mitigation, restoration, and learning capacities reduce risk?

All values in the sample workflows are illustrative. Replace them with documented climate scenarios, biodiversity indicators, demographic projections, planetary-boundary status data, governance assessments, and transparent assumptions before applied use.
