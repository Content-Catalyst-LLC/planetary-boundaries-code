# Biogeochemical Flows: Nitrogen, Phosphorus, and Planetary Destabilization

This article folder supports the Planetary Boundaries knowledge-series article on biogeochemical flows, nitrogen, phosphorus, nutrient surplus, eutrophication, dead zones, nutrient-use efficiency, watershed connectivity, and planetary destabilization.

The article body includes Python and R workflows for readability and reproducible analytics. This repository extends the implementation into a broader engineering scaffold for auditability, APIs, systems programming, embedded monitoring, TinyML, and PYNQ-oriented edge workflows.

## Structure

- `python/` — nitrogen-phosphorus boundary diagnostics
- `r/` — nutrient boundary dashboarding
- `sql/` — schema for nutrient inputs, uptake, runoff, ecosystem sensitivity, and audit trails
- `rust/` — reliable nutrient-risk scoring-core scaffold
- `go/` — lightweight diagnostic API scaffold
- `c/` — embedded-style nitrate/phosphate threshold alert scaffold
- `cpp/` — nutrient policy scenario simulation scaffold
- `tinyml/` — edge anomaly detection scaffold
- `pynq/` — accelerator-oriented monitoring scaffold
- `docs/` — methodological and engineering notes
- `data/raw/` — raw input data
- `data/processed/` — cleaned and transformed data
- `outputs/` — exported diagnostic scores and dashboard-ready files

## Methodological Note

The sample workflows use illustrative data. Replace placeholder values with documented fertilizer records, manure estimates, crop uptake data, watershed monitoring, hydrological data, wastewater records, ecological sensitivity estimates, and transparent assumptions before applied use.
