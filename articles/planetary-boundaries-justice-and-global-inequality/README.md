# Planetary Boundaries, Justice, and Global Inequality

This article folder supports the Planetary Boundaries knowledge-series article on justice, global inequality, ecological overuse, minimum access, vulnerability, differentiated responsibility, and planetary stewardship.

The article body includes Python and R workflows for readability and reproducible analytics. This repository extends the implementation into a broader engineering scaffold for auditability, APIs, systems programming, embedded monitoring, TinyML, and PYNQ-oriented edge workflows.

## Structure

- `python/` — planetary justice and inequality scoring
- `r/` — justice, access, and boundary-pressure dashboarding
- `sql/` — schema for groups, ecological use, access indicators, vulnerability, and audit trails
- `rust/` — reliable justice-gap scoring-core scaffold
- `go/` — lightweight diagnostic API scaffold
- `c/` — embedded-style threshold alert scaffold
- `cpp/` — scenario simulation scaffold
- `tinyml/` — edge anomaly detection scaffold
- `pynq/` — accelerator-oriented monitoring scaffold
- `docs/` — methodological and engineering notes
- `data/raw/` — raw input data
- `data/processed/` — cleaned and transformed data
- `outputs/` — exported diagnostic scores and dashboard-ready files

## Methodological Note

The sample workflows use illustrative data. Replace placeholder values with documented environmental accounts, social indicators, vulnerability metrics, historical responsibility data, and transparent allocation methods before applied use.
