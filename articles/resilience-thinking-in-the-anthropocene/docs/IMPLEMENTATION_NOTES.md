# Implementation Notes

This module is designed around five linked questions:

1. How much boundary pressure is a social-ecological system experiencing?
2. How much ecological buffering remains through function, diversity, and redundancy?
3. How strong are adaptive, learning, governance, and justice capacities?
4. Where is resilience adaptive, fragile, maladaptive, or transformation-oriented?
5. How can resilience diagnostics become auditable and reproducible?

All values in the sample workflows are illustrative. Replace them with documented ecological indicators, social vulnerability data, monitoring records, governance assessments, and transparent assumptions before applied use.
