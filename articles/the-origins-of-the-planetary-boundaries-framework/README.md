# The Origins of the Planetary Boundaries Framework

This article folder supports the Planetary Boundaries knowledge-series article on the origins of the planetary boundaries framework, including Earth system science, Anthropocene thinking, resilience theory, safe operating space, the 2009 formalization, the original nine boundaries, the 2015 refinement, later assessments, governance uptake, critique, justice interpretation, and framework diffusion.

The article body includes Python and R workflows for readability and reproducible analytics. This repository extends the implementation into a broader engineering scaffold for auditability, knowledge graphs, APIs, structured literature review workflows, policy-diffusion tracking, and framework-evolution diagnostics.

## Structure

- `python/` — framework evolution and influence diagnostics
- `r/` — framework evolution dashboarding
- `sql/` — schema for milestones, source documents, revisions, concepts, and audit trails
- `rust/` — reliable framework-evolution scoring-core scaffold
- `go/` — lightweight diagnostic API scaffold
- `c/` — simple milestone threshold alert scaffold
- `cpp/` — framework-diffusion scenario simulation scaffold
- `tinyml/` — optional source-triage anomaly scaffold
- `pynq/` — optional accelerated document-processing scaffold
- `docs/` — methodological and engineering notes
- `data/raw/` — raw input data
- `data/processed/` — cleaned and transformed data
- `outputs/` — exported diagnostic scores and dashboard-ready files

## Methodological Note

The sample workflows use illustrative scoring. Replace placeholder values with bibliometric records, source documents, policy references, structured literature review coding, citation data, and transparent assumptions before applied use.
