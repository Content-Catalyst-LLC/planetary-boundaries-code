# Earth System Governance in an Age of Limits

This article folder supports the Planetary Boundaries knowledge-series article on Earth system governance in an age of limits.

The article body includes Python and R workflows for readability and reproducible analytics. This repository extends the implementation into a broader engineering scaffold for auditability, APIs, systems programming, embedded monitoring, TinyML, and PYNQ-oriented edge workflows.

## Structure

- `python/` — governance-fit and boundary-risk scoring
- `r/` — cross-scale governance dashboarding
- `sql/` — schema for governance cases, boundary domains, institutional capacity, and audit trails
- `rust/` — reliable governance-fragility scoring-core scaffold
- `go/` — lightweight diagnostic API scaffold
- `c/` — embedded-style boundary alert scaffold
- `cpp/` — scenario simulation scaffold
- `tinyml/` — edge anomaly detection scaffold
- `pynq/` — accelerator-oriented monitoring scaffold
- `docs/` — methodological and engineering notes
- `data/raw/` — raw input data
- `data/processed/` — cleaned and transformed data
- `outputs/` — exported diagnostic scores and dashboard-ready files

## Methodological Note

The sample workflows use illustrative data. Replace placeholder values with documented governance indicators, legal analysis, institutional data, expert elicitation, participatory scoring, or boundary-specific monitoring systems before applied use.
