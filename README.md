# planetary-boundaries-code

Code workflows, analytical models, and implementation scaffolding for the Planetary Boundaries knowledge series.

## Structure
- articles/            article-specific code and notes
- python/              shared Python workflows
- r/                   shared R workflows
- sql/                 optional schemas and data structures
- data/raw/            raw input data
- data/processed/      processed datasets
- outputs/             exported results
- docs/                series-level implementation notes
